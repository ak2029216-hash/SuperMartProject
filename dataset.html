Dataset Overview

The dataset contains historical retail sales information used to train and evaluate machine learning models for sales prediction.

Feature Description
Item Identifier Unique identifier of each product
Item Weight Weight of the product
Item Fat Content Low Fat or Regular Fat category
Item Visibility Display area percentage in store
Item Type Category of product
Item MRP Maximum Retail Price
Outlet Identifier Store identification number
Outlet Size Small, Medium or High size outlet
Outlet Type Supermarket or Grocery Store
Item Outlet Sales Target variable used for prediction
Dataset Highlights
  • 8523 sales records
  • Retail store information
  • Product attributes
  • Outlet characteristics
  • Historical sales values
  • Used for machine learning forecasting

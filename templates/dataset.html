Core Attribute Schema Mapping

Attribute Column Name Data Type Functional Profile Definition
Item_Identifier Alphanumeric Unique stock-keeping unit (SKU) product tracking identification key code.
Item_Weight Continuous Float Physical weight metric of the product item lines (Imputed via column values median).
Item_Fat_Content Categorical Dietary descriptor mapping the inventory fat footprints (Low Fat or Regular).
Item_Visibility Continuous Float The percentage allocation of store layout on-shelf space dedicated to a product.
Item_Type Categorical Groups specific product ranges across 16 distinct retail categories (e.g., Snacks, Dairy).
Item_MRP Continuous Float Maximum Retail Price (List Price) used to scale baseline revenue forecasting functions.
Outlet_Identifier Alphanumeric Unique storefront structural registration key code mapping across 10 separate outlets.
Outlet_Establishment_Year Discrete Integer The foundational year code transformed into a dynamic operational Outlet_Age variable.
Outlet_Size Categorical Ordinal The operational footprint layout scale classification categorized as Small, Medium, or High.
Outlet_Location_Type Categorical Regional geographic topology tier classifications separating stores (Tier 1, 2, or 3).
Outlet_Type Categorical Foundational corporate business model (Grocery Store vs Supermarket Type 1, 2, or 3).
Item_Outlet_Sales Dependent Continuous The target variable indicating total regional transactional revenues generated by a SKU.
